Shared UI components.

## Features

- Desktop / mobile detection.
- Fonts.
- Avatar.
- Human date and time representation.

## Getting started

In _pubspec.yaml_:
```yaml
dependencies:
  zxbase_flutter_ui: ^1.0.0
```

In your code:
```
import 'package:zxbase_flutter_ui/zxbase_flutter_ui.dart';
```
